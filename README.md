Цель проекта — определить ключевые потребности и болевые точки клиентов компании по продаже мебели на основе анализа текстовых отзывов. Результаты должны помочь бизнесу улучшить продукт, сервис и стратегию взаимодействия с клиентами.
Ссылка на датасет: https://disk.yandex.ru/d/OSpR6J2dZT1cRQ
Гипотезы для проверки (5 штук)
	Клиенты чаще оставляют негативные отзывы из-за уровня сервиса, а не из-за качества мебели.
	Высокая скорость доставки значительно повышает вероятность получения положительной оценки.
	Наиболее частая причина возвратов и претензий — несоответствие товара описанию.
	Покупатели мягкой мебели (диваны, кресла) оставляют более эмоциональные и длинные отзывы по сравнению с покупателями корпусной мебели.
	Чем выше цена товара, тем более подробный отзыв оставляет клиент.
